# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview

This is a beginner-friendly Machine Learning project that predicts house prices based on the size of the house using **Linear Regression**.

The project demonstrates the complete ML workflow:

* Data creation
* Data preprocessing
* Model training
* Prediction

---

## 🚀 Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-learn

---

## 📊 Dataset

The dataset is a simple manually created dataset:

| Size (sq ft) | Price |
| ------------ | ----- |
| 500          | 100   |
| 1000         | 200   |
| 1500         | 300   |
| 2000         | 400   |
| 2500         | 500   |

---

## 🧠 How it Works

* The model learns the relationship between **Size** and **Price**

* It identifies a linear pattern:

  👉 Price ≈ Size / 5

* Using this relationship, it predicts prices for new inputs
